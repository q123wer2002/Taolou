架構
======
1. 主要分成 css, js, images, font這幾個資料夾，分別放該放的東西
2. 頁面製作主要以當初ppt上面的function頁面製作
3. 有一個layout.html，主要是板型的html
4. 每個檔案都會用到的東西都放在layout裡面，像layout.css, layout.js等等
5. 寫的html主要以清晰，擴充性高，易閱讀為主
